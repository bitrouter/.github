# BitRouter

**Agent-native LLM router that optimizes your agent with every run.**

No agent changes needed — drop BitRouter in front of your existing setup and every model call becomes reliable, traceable, secure, and cost-effective. The longer it runs, the smarter it routes.

Free to self-host. Cloud option available.
